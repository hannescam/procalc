# procalc
- a simple app for linux that can calculate percentages.
- a windows exe file is in the releases and also a linux binary
- to compile: 
- 		make build
- to clean: 
- 		make clean
- to install: 
- 		sudo make install
- to uninstall: 
- 		sudo make uninstall
